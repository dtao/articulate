Articulate

    What dreams are made of

    This is a true WYSIWYG editor. You're editing HTML directly.

    As you type, the table of contents on the left is updated dynamically.

    Ctrl+1
    sets the current element to a top-level heading
    Ctrl+2
    sets the current element to a subheading
    Ctrl+3
    sets the current element to a secondary subheading
    Ctrl+P
    sets the current element to a regular paragraph
    Ctrl+L
    converts the current element to a list
