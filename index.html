This is a true WYSIWYG editor. You're editing HTML directly. The styles on this page are all available and editable.

Keyboard shortcuts

  • Ctrl+1 sets the current element to a top-level heading (an <H1> element).
  • Ctrl+2 sets the current element to a subheading (an <H2> element).
  • Ctrl+3 sets the current element to a secondary subheading (an <H3> element).
  • Ctrl+P sets the current element to a regular paragraph.
  • Ctrl+L converts the current element to a list.
